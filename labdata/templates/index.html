A virtual laboratory

This "lab" is a collection of analytical data amassed by Daven Quinn during various projects, stored in the EarthCube Lab Data Interface system. It serves to demonstrate the flexibility of this system for storage and interaction with a variety of data types collected in the Earth sciences.

This lab's data management system conforms to the EarthCube Lab Data Interface data storage and API specification, version 0.1-alpha.
